Exemplo 1

No exemplo 1, temos uma página web que recebe um cep e após a requisição, irá preencher com o resposta da requisição os campos de rua, bairro, cidade. Para que isso ocorra, quando o usuário estiver preenchendo o cep, o JavaScript já é executado realizando a validação do tamanho o cep, caso o cep tenha o tamanho 9 será chamado a função XMLHttpRequest.

Quando ocorrer a chamada da requisição, será passado o parâmetro busca-endereco.php com o método get passando o cep na url que será executado assincrôno.

No script php, será realizado a validação do cep de acordo com os endereços pré-cadastrados e caso o cep bater com que tem será retornado o endereço no formato JSON.

Quando o retorno do script php ocorrer, vamos verificar se o retorno foi sucesso e acessar o JSON convertendo o JSON para um objeto JavaScript e depois manipular a árvore DOM via javaScript e inserir as informações contidas no JSON nos inputs.

Exemplo 2

O processo será parecido com a do exemplo 1, a diferença é que estamos definindo a propriedade responseType e dessa forma não será necessário converter o retorno para um objeto JavaScript porque já estamos resgatando a resposta em um formato específico.

Exemplo 3

Realiza o mesmo processos dos exemplos anteriores, a diferença está na parte que definimos o cabeçalho HTTP 'Content-Type' para envio do JSON e envia a requisção convertida do objeto JS para JSON usando JSON.stringfy.
